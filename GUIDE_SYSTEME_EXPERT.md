# Guide d'Utilisation du Système Expert de Remplacement

## Pour les Cadres de Santé

### Comment utiliser le système ?

#### 1. Accéder à une absence

1. Connectez-vous en tant que cadre
2. Allez dans le menu "Absences"
3. Cliquez sur une absence à traiter

#### 2. Vérifier la disponibilité

1. Cliquez sur le bouton **"Vérifier disponibilité"**
2. Le système analyse automatiquement tous les agents du service
3. Attendez quelques secondes (un spinner s'affiche)

#### 3. Comprendre les résultats

##### Si des suggestions sont trouvées ✅

Vous verrez:
- **Un résumé** en haut: nombre de candidats analysés
- **Des cartes de suggestions** classées par score (du meilleur au moins bon)
- **Des recommandations** pour vous guider

Chaque suggestion affiche:
- 🥇 **Rang** (1er, 2ème, 3ème...)
- **Nom** de l'agent
- **Score** (plus c'est élevé, mieux c'est)
- **Points forts** (✅ en vert):
  - Disponibilité inscrite
  - Même service
  - Droits à congés disponibles
  - etc.
- **Points d'attention** (⚠️ en orange):
  - Charge de travail élevée
  - Plusieurs remplacements récents
  - etc.

##### Si aucune suggestion ❌

Vous verrez:
- **Explication** du pourquoi (conflits, absences)
- **Solutions alternatives**:
  - Bouton "Rechercher un intérimaire"
  - Bouton "Rechercher via Hublo"
- **Recommandations** pour débloquer la situation

#### 4. Discuter avec l'IA

Un chatbot intelligent est disponible en bas de page:

1. Cliquez sur **"Discuter avec l'IA"** pour l'ouvrir
2. Posez vos questions en français
3. L'IA vous répond avec des explications détaillées

**Questions que vous pouvez poser:**

- "Quelles sont les règles?"
- "Pourquoi ce candidat est premier?"
- "Comment est calculé le score?"
- "Que faire si aucun remplaçant?"
- "Comment vérifiez-vous les disponibilités?"
- "Comment assurez-vous l'équité?"
- "Quelle est l'importance de la spécialité?"

#### 5. Sélectionner un remplaçant

1. Cliquez sur **"Sélectionner"** sur la carte de suggestion
2. Ou utilisez le menu déroulant pour choisir manuellement
3. Cliquez sur **"Appliquer"** pour valider

## Comprendre les Scores

### Score élevé (>100) ✅
- **Excellent candidat**
- Disponibilité inscrite
- Même service et spécialité
- Droits disponibles
- Charge légère
- **Action**: Sélectionner en priorité

### Score moyen (50-100) ⚠️
- **Bon candidat avec réserves**
- Disponibilité partielle ou pas inscrite
- Quelques points d'attention
- **Action**: Vérifier les détails avant de sélectionner

### Score faible (<50) ❌
- **Candidat peu recommandé**
- Peu de points forts
- Plusieurs points d'attention
- **Action**: Privilégier d'autres options

## Cas d'Usage Fréquents

### Cas 1: Absence courte (1-2 jours)

**Situation**: Un agent est absent 2 jours

**Le système:**
1. Cherche les agents avec disponibilité inscrite
2. Privilégie ceux avec charge légère
3. Vérifie les droits disponibles

**Votre action:**
- Sélectionnez le premier suggéré si score >100
- Vérifiez les points d'attention si score <100

### Cas 2: Absence longue (1-2 semaines)

**Situation**: Un agent est absent 10 jours

**Le système:**
1. Cherche une couverture complète en priorité
2. Propose une couverture partielle si nécessaire
3. Vérifie la charge mensuelle

**Votre action:**
- Privilégiez "Couverture complète" si disponible
- Sinon, combinez plusieurs agents pour couvrir la période

### Cas 3: Aucun remplaçant disponible

**Situation**: Tous les agents sont occupés ou absents

**Le système:**
1. Explique pourquoi (conflits listés)
2. Propose des solutions alternatives

**Votre action:**
1. Cliquez sur "Rechercher un intérimaire" (email pré-rempli)
2. Ou cliquez sur "Rechercher via Hublo"
3. Ou contactez manuellement les agents
4. Ou ajustez les dates de l'absence

### Cas 4: Plusieurs options équivalentes

**Situation**: 3-4 agents avec des scores proches

**Le système:**
1. Applique les règles d'équité
2. Favorise ceux avec moins de remplacements récents

**Votre action:**
- Comparez les profils
- Privilégiez l'équité (moins de remplacements récents)
- Considérez les préférences personnelles si connues

## Bonnes Pratiques

### ✅ À faire

1. **Vérifier la disponibilité** avant de valider une absence
2. **Lire les points d'attention** avant de sélectionner
3. **Utiliser le chatbot** si vous avez des questions
4. **Privilégier l'équité** entre les agents
5. **Considérer la charge de travail** mensuelle

### ❌ À éviter

1. Ne pas vérifier les suggestions avant de valider
2. Toujours sélectionner le même agent
3. Ignorer les avertissements (⚠️)
4. Ne pas utiliser les solutions alternatives si aucun remplaçant
5. Surcharger les agents avec trop de remplacements

## Questions Fréquentes

### Q: Pourquoi le système ne suggère personne ?
**R**: Tous les agents sont soit déjà planifiés, soit absents pendant la période. Utilisez les solutions alternatives (intérimaire/Hublo) ou ajustez les dates.

### Q: Puis-je sélectionner un agent non suggéré ?
**R**: Oui, le menu déroulant affiche tous les agents du service. Mais attention aux conflits potentiels.

### Q: Comment le système assure l'équité ?
**R**: Le système pénalise les agents qui ont déjà fait beaucoup de remplacements récents et privilégie ceux avec une charge légère.

### Q: Que signifie "couverture partielle" ?
**R**: L'agent est disponible sur une partie seulement de la période d'absence. Vous devrez trouver un autre remplaçant pour le reste.

### Q: Les scores sont-ils fiables ?
**R**: Oui, ils sont calculés selon 9 règles métier validées. Mais vérifiez toujours les détails avant de décider.

### Q: Puis-je modifier les règles ?
**R**: Non en tant que cadre. Contactez l'administrateur système si vous pensez qu'une règle doit être ajustée.

## Support

### En cas de problème

1. **Vérifiez la console** (F12) pour les messages d'erreur
2. **Utilisez le chatbot** pour poser des questions
3. **Contactez l'administrateur** si le problème persiste

### Retours et Suggestions

Vos retours sont précieux pour améliorer le système:
- Quelles règles fonctionnent bien ?
- Quelles règles devraient être ajustées ?
- Quelles nouvelles fonctionnalités souhaitez-vous ?

Contactez l'équipe technique pour partager vos retours.
